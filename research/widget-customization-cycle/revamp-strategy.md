# Revamp Strategy

## Product thesis
StudyPlanner should not become a generic Widgetsmith clone. The winning product is: **syllabus-to-plan utility with premium home-screen personalization powered by real schoolwork.**

## Highest leverage implementation set
1. Keep Scan/Review/Planner data integrity untouched.
2. Make Widgets a primary tab with flagship Widget Studio treatment.
3. Make native Today/Upcoming widgets free and truthful.
4. Add advanced Plus templates as previews/presets, not fake native claims.
5. Add deep links so widget taps and QA can route directly to Widget Studio.
6. Preserve IAP IDs and App Review compliance.

## Why not a blank-canvas editor now
A blank editor is high effort, high support burden, and risky for App Review if previews imply unsupported native behavior. Constrained templates deliver more student value faster.

## QA standard
- Real iPhone Simulator screenshots only.
- Typecheck must pass.
- Release checks preferred; missing npm scripts are documented, not hidden.
- Any demo data must be clearly labeled.
