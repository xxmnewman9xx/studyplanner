# High-Leverage UI Patterns

1. Liquid glass cards with depth, translucent surfaces, and soft borders.
2. One primary next action per screen.
3. Live dashboard tiles instead of static empty states.
4. Customization during onboarding to create ownership.
5. Widget previews tied to planner data.
6. Calm paywall with no fake urgency.
7. Notes/schedule/classwork as one student OS, not separate utilities.
8. Larger titles, generous line heights, fewer truncation traps.
9. Gen Z customization through themes without visual chaos.
10. App Store screenshot-ready hero sections.
