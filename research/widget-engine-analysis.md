# Widget Engine Analysis

Widgetsmith-style systems win because they combine:
- template gallery
- visual theme packs
- live preview
- saved presets
- meaningful default data
- identity/personalization

StudyPlanner should not become a blank-canvas widget editor. The winning wedge is school-specific widgets: Today, Upcoming, Schedule, Countdown, Class Risk, Focus Block, and Smart Stack moments backed by real planner data.
