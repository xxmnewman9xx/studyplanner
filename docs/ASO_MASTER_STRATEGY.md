# ASO Master Strategy

## Recommended metadata

- App name: Study Planner: Syllabus AI if AI parser proof exists; fallback Study Planner: Syllabus.
- Subtitle: Scan syllabi. Track deadlines.
- Keywords: homework,assignments,exams,calendar,classes,grades,school,college,student,widget,reminders,timetable
- Promotional text: Add a syllabus, review found work, and keep confirmed deadlines visible in Today, Week, Calendar, Classes, and Home Screen widgets.
- Verification: `npm run verify:aso` writes `docs/ASO_COPY_AUDIT.md` and currently passes English field limits plus unsafe-claim checks. This is not App Store Connect acceptance or localized/native review.

## Positioning

Lead with the strongest truthful loop: syllabus/materials -> review found work -> confirmed plan -> Today/Week/Calendar/Classes/widgets.

## Keyword clusters

Core planner: student planner, school planner, homework planner, assignment tracker.  
Syllabus: syllabus planner, syllabus scanner, scan syllabus, syllabus AI if endpoint-backed.  
Calendar/reminders: deadline tracker, exam calendar, homework reminders.  
Widgets: homework widget, student widget, assignment widget.  
Parents: homework tracker for kids, middle school planner.

## Claims to avoid

Perfect extraction, guaranteed grades, never miss, every syllabus, unsupported LMS sync, unsupported widget families, and unverified free trials/lifetime offers.
