# External Proof Packet Templates

These files are templates only. Do not rename a template to the proof filename until the evidence is real, current, and reviewed.

`npm run verify:submission` expects real proof files in this folder:

- `storekit-sandbox-proof.md`
- `app-store-connect-screenshot-upload.md`
- `signed-archive-entitlements.md`
- `voiceover-traversal.md`
- `localized-ui-native-review.md` only when submitting localized metadata/screenshots with `STUDYPLANNER_SUBMIT_LOCALIZATIONS=1`

The verifier deliberately rejects files that still contain words such as `TEMPLATE`, `TODO`, `placeholder`, `replace before submit`, or `sample only`.
